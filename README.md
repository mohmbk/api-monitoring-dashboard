# API Monitoring Dashboard

An API Monitoring Dashboard built with **Go**, **React**, **TypeScript**, **MongoDB**, and **JWT Authentication**. The application allows users to monitor the availability and performance of their APIs through a modern and intuitive dashboard.

## Features

- 🔐 Secure user authentication with JWT
- ➕ Add and manage API endpoints
- 🗑️ Delete monitored APIs
- 📊 Display the latest API status
- ⏱️ Monitor API response times
- 🌐 Track HTTP status codes
- 📜 View the complete history of API health checks
- 🔄 Automatic API monitoring every 60 seconds

## Tech Stack

### Frontend
- React
- TypeScript
- React Router
- CSS

### Backend
- Go (Golang)
- net/http
- MongoDB
- JWT Authentication
